## 小游戏系统
### 体育系统
- 地点：武东操场
- 耗费：5天/回合
- 获得：SportScore,SportTime
- 游戏数值
    - 长跑
    - 短跑
    - 引体向上
    - 肺活量

 |项目 |分值 | | | |

 |长跑| score >=22 | 22> score >= 20 | 20> score >=18 | 18> score |
 |短跑| score <=6 | 7>= score > 6 | 8>= score >7 |  score > 9 |
